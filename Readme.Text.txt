# HR ANALYTICS: EMPLOYEE ATTRITION DEEP DIVE
## Project Overview
Analyzed 1470 employees to identify What is attrition rate? Which department loses
the most people? Does salary affect attrition? Does overtime cause people to quit?
## Tools Used
- Excel (data cleaning)
- MySQL (data storage and SQL analysis)
- Power BI( Charts and report generation)
## Key Findings
Total Departures: 237 total attrition count
Research & Development drives the most exits, accounting for 65.4% of all departures.
Low-salary range  employees experience the highest attrition rate at 21.8%.
